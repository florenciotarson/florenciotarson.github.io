# Privacy Policy

**Last updated: March 31, 2026**

This Privacy Policy describes how the Florenciotarson Discord application ("the App") handles data.

## Data Processed

The App may process Discord-related data only as needed for its functionality. This includes:

- Discord user IDs
- Discord server IDs
- Discord channel IDs
- Messages or command content explicitly sent to the bot
- Basic configuration data

## How Data Is Used

Data is used solely to operate bot features, manage settings, and ensure the security and reliability of the App. No personal data is sold to third parties.

## Data Sharing

Data may be shared only if required by law or if necessary for infrastructure and service operation.

## Data Retention

Data is retained only as long as necessary to provide the App's functionality.
